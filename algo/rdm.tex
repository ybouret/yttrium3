\section{Description}

Let's assume we have a cluster of $M$ active species and $N$ equilibria:
\begin{equation}
\forall i, E_i : \sum_j \nu_{ij} A_j = 0 : K_i=\prod_j C_j^{\nu_{ij}}
\end{equation}
We define the topology matrix:
\begin{equation}
\bm{\nu} = \begin{bmatrix} \nu_{ij} \end{bmatrix} \in \mathcal{M}_{N,M}
\end{equation}
Starting from $\vec C_0\in \mathbb{R}^M$, the reachable state by the extent vector $\vec \xi \in \mathbb{R}^N$ is:
\begin{equation}
	\vec C = \vec C_0 + \bm{\nu}^T \vec \xi 
\end{equation}


\section{Conservation Laws}

\subsection{Computation}
A conservation law is non-trivial, positive linear combination of species that is not affected by an extent.
They appear in the {\bf CoVen} algorithm of $\bm{\nu}^\perp$. If a law is found, then the involved species are conserved.
 
\subsection{Algebraic Form}
Each contraint is a row vector $\vec \Gamma_i$ of:
\begin{equation}
\bm{\Gamma} \in \mathcal{M}_{Nc,M},\;\;\mathrm{rank}(\bm \Gamma) = r_g \leq 
\end{equation}
Any modification orthogonal to $\bm \nu$ from $\vec C_0$ is:
\begin{equation}
\vec{C} = \vec C_0 + \bm{\Gamma}^T \vec\varphi,\;\; \vec \varphi \in \mathbb{R}^{N_c}
\end{equation}
